Welcome to Strat Page Info

This page is meant to test the theme switching functionality and demonstrate how the CSS variables can be used to create different looks.

Details

The theme selector in the navbar allows you to switch between three different themes: Light, Dark, and Sunset. Each theme is defined using CSS variables that control colors, shadows, and other visual properties.

Customization

You can customize the themes by editing the CSS variables in the stylesheet located at static/css/info.css. This allows you to create your own unique look for the page.
